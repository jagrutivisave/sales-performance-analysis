# Sales Performance & Revenue Optimization Analysis

## Objective
Analyze sales data to identify revenue trends, top-performing products, regions, and customers, and provide business insights for decision-making.

## Tools Used
- SQL
- Microsoft Excel
- Power BI

## Key Analysis
- Sales trend analysis
- Revenue and growth analysis
- Product and region performance
- Customer analysis

## Business Insights
- Identified high and low performing regions
- Highlighted top products contributing to revenue
- Provided insights to improve sales performance
